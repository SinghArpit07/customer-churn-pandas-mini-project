# customer-churn-pandas-mini-project

## Data Ingestion

This project downloads the Telco Customer Churn dataset from Kaggle using the Kaggle API.

### Prerequisites
- Python 3.11
- Kaggle API credentials (`kaggle.json`)

### Steps
1. Create a virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Download data:
   python scripts/download_data.py
